周二上午，大家在硅谷小镇客厅的会议室迎来了「[个体商业变现](https://mp.weixin.qq.com/mp/appmsgalbum?action=getalbum&__biz=MzYyNDI1NDY4Mg==&scene=2&album_id=4252307827437256712&count=3#wechat_redirect)」系列活动的首次收费增强版深度工作坊。

这场「品牌内核锻造工作坊」干货满满，溢出了原定的两小时时间～

![](01.jpg)

当人员到齐后，因为有相互不认识的，各自极其简略地说了下自己是谁，至于做的事业会在之后的互动练习中展现出来。

显宁老师先带大家回顾了[第一场免费沙龙](https://mp.weixin.qq.com/s/vwYDkyOWIZN4jm9C0uQGaA)中提到的商业活动四环节：流量 → 销售 → 交付 → 跟进。

接着，又提出通过个人品牌变现的四个环节：洞察 → 萃取 → 转化 → 交付。

其中，个人品牌变现的「转化」对应商业活动的「销售」，而「交付」则对应商业活动的「交付」与「跟进」。

![](02.jpg)

在讲到「洞察」时，一位参与者对于自己事业是否处于风口以及目标群体付费意愿方面在自己的觉察之上有些许疑问。

老师分享了如何更为客观地判断与佐证某个方向、某件事是否位于风口，还有让目标群体付费的大致方法。

![](03.jpg)

上面是向外洞察的部分，也就是洞察需求、市场，接下来就到了向内洞察——见自己。

虽说在以往几场免费沙龙中也有「见自己」，但那都是参与者完全靠自己去想，会比较浅一些。

这场深度工作坊与之不同，活动前一天，老师在群中抛出了几个精心设计的问题让大家提前思考——在引导之下，能够更深入地从经历中探寻自己的本质。

大家在分享各自对那几个问题的思考与回答后，纷纷表示自己想通了一些事，看到了未曾想过的优势与不足。

![](04.jpg)

在讲了怎样萃取自己的品牌与产品或服务，并稍带着说了些「转化」后，这场超出原定时长的深度工作坊结束了。

很明显，收费的深度工作坊与免费的沙龙相比，虽然核心的东西都差不多是一样的，很少且很简单（大道至简），但讲解的全面性、精细度有很大差别。
